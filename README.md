# NodeJS CRUD sample (post)

**Run NodeJS App**
1. create .env copy from env-example
2. cd NodeJS-sample
3. npm install
4. npm run start
5. access api from localhost:8080

**Routes**
Route | Method | Body
| :--- | ---: | :---:
/api/posts | GET | \| 
/api/posts  | POST | { title: string, description: string }
